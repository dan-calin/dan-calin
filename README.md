## Dan Calin - About Me 👋

> Building real things **with** AI — and learning to build real **AI**.

I'm a developer exploring where modern software meets AI. I use AI to ship faster,
and increasingly I build AI *into* the things I make — voice agents, LLM tooling,
and automation. This profile is a running log of that progress: improving **how I
work with AI**, while growing the engineering skills underneath it.

- 🌱 **Currently exploring:** LLM-powered apps, voice agents & tool-calling, ML-driven automation
- 🧪 **Approach:** ship small, real projects → learn what breaks → go deeper
- 💬 **Open to:** opportunities, collaboration, and a good technical conversation

### 🤖 How I work with AI

I treat AI as a pair-programmer and a force-multiplier — not autopilot. Two threads
run through my projects:

1. **Building *with* AI** — using AI tools to design, scaffold, debug and refactor
   faster, while staying in the driver's seat on architecture and trade-offs.
2. **Building *AI in*** — wiring real LLM capabilities into products: voice I/O,
   tool-calling, retrieval, and agents that actually *do* things on your machine.

Every project here taught me something the last one didn't — and that's the point.
